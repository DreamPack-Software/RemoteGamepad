## ✨ Key Features

📶 Wireless Gamepad
---
Turn your phone or tablet into a real-time controller

Works seamlessly with supported games and emulators

Low-latency input ensures smooth gameplay

🎛️ Customizable Layouts
---
Full control over button placement and joystick positions

Save multiple profiles for different games

Adjust sensitivity and responsiveness per layout

🧠 Multi-Platform Support
---
Compatible with Windows PC games

Works with popular emulators

Integrates with platforms supporting virtual controller input

🔌 Connection Compatibility
---
PC Receiver/Server app (primary method for stable input)

USB tethered connection for direct control

Local pairing between mobile device and host system

Virtual controller input via host software

Bluetooth is supported! Means you don't need Wi-Fi or USB at all!

🎯 Common Use Cases
---
Play PC games without a physical controller

Multiplayer sessions using extra mobile gamepads

Emulation gaming with custom layouts

Streaming setups requiring portable controller input

Rapid setup for casual or competitive gaming

🧠 Design Philosophy
---
Remote Gamepad focuses on simplicity, responsiveness, and flexibility 🔥
It doesn’t require complicated drivers or hardware — just install, connect, and play. The app gives users full control over layout, sensitivity, and connection, while keeping setup clean and straightforward.

📦 Notes
---
Requires PC receiver for proper input

USB and local network connections are supported

Experimental Bluetooth support is not reliable

Optimized for low-latency gaming experiences
